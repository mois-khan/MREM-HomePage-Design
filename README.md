# MREM PAGE
A professional landing page for **Malla Reddy Engineering College & Management Sciences (MREM)**.

## About the Project
Malla Reddy Engineering College & Management Sciences (MREM) is a premier institution affiliated with JNTUH and accredited by NAAC & AICTE. This project is a web-based landing page designed to showcase the college's academic programs, mission, and facilities.

## Features
- **Modern UI**: Clean and responsive design.
- **Course Information**: Detailed list of B.Tech, MBA, and MCA courses.
- **Mission & Vision**: Clear presentation of the college's goals.
- **Admissions Section**: Call-to-action for prospective students.
- **Interactive Navigation**: Easy access to different sections of the page.

## Technologies Used
- HTML5
- CSS3 (Custom styling in `style.css`)
- JavaScript (Interactive features in `script.js`)

## Setup
To run this project locally, simply clone the repository and open `index.html` in your browser.

```bash
git clone https://github.com/[YOUR-USERNAME]/MREM-PAGE.git
cd MREM-PAGE
# Open index.html in a browser
```
